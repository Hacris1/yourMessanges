# yourMessanges

Una aplicación de mensajería segura y cifrada, desarrollada con tecnologías modernas de frontend y backend, enfocada en la privacidad y seguridad de las comunicaciones mediante criptografía RSA.

## 🌐 Aplicación Desplegada

- **Frontend**: https://messanges-hfs.onrender.com
- **Backend**: https://yourmessanges.onrender.com

## 📋 Descripción del Proyecto

**yourMessanges** es una plataforma de mensajería privada que implementa cifrado end-to-end usando RSA. El proyecto está dividido en dos partes principales:

### Backend
- Desarrollado con **TypeScript** y **Node.js**
- Gestiona autenticación, almacenamiento de mensajes y comunicación en tiempo real via WebSockets
- Implementa encriptación RSA para proteger los datos sensibles
- API REST para operaciones CRUD de usuarios y mensajes

### Frontend
- Construido con **React + TypeScript** y **Vite**
- Interfaz intuitiva para registrarse, autenticarse y enviar mensajes cifrados
- Gestión de claves RSA y derivación de contraseñas
- Comunicación en tiempo real con WebSockets

## 🔐 Características de Seguridad

- ✅ Autenticación de usuarios
- ✅ Encriptación RSA end-to-end
- ✅ Comunicación cifrada mediante WebSockets
- ✅ Derivación de contraseñas segura
- ✅ Regeneración de claves criptográficas

## 🚀 Tecnologías Utilizadas

### Backend
- TypeScript

- Express

- Socket.io

### Frontend
